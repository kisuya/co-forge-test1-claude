# Project project-003: history-sharing-community

## Goal
서비스 완성 — P0 품질 보강부터 이벤트 히스토리, 리포트 공유, 커뮤니티, 데일리 브리핑, 뉴스 피드, 트렌딩, 마켓 캘린더까지 전 기능 구현. 프로덕션 수준의 레이아웃, SEO, 접근성, PWA를 갖춘 완성된 제품으로 전환.

## Context
project-001에서 P0 전체(인증, 종목관리, 급변동감지, AI리포트, 대시보드), project-002에서 P1 전체(UX보강, 알림, 미국주식, 유사케이스)를 완료. 핵심 기능은 모두 동작하나, 시세 미연동, CORS+에러 처리 미흡, 종목 관리 UX 부족 등이 남아있음. 시장 조사를 통해 경쟁사 대비 필요한 기능(데일리 브리핑, 뉴스 피드, 트렌딩, 마켓 캘린더)을 추가 도출. 이번 프로젝트는 3개월 집중 개발로 남은 모든 기능을 구현하고, 프로토타입이 아닌 프로덕션 수준의 완성된 서비스로 만드는 것이 목표.

## Scope
- **품질 & 기반 (8건)**: CORS 에러 처리, API 에러 표준화, DB 인덱싱, Redis 캐싱, 레이트리밋, 입력 검증, 구조화 로깅
- **시세 데이터 & StockCard (6건)**: 시세 API 연동, 가격 표시 UI, 추적 현황, ⋮ 관리 메뉴(데스크톱+모바일)
- **이벤트 히스토리 (6건)**: 히스토리 API, 종목 상세 페이지, 타임라인 UI, 페이지네이션, 빈 상태, 네비게이션
- **리포트 공유 (5건)**: 공유 모델, URL 생성 API, 비인증 조회, 공유 UI, 공유 페이지+만료
- **마이페이지 (6건)**: 닉네임 모델+API, 통계 API, 프로필 UI, 닉네임 편집, 비밀번호 변경, 활동 이력
- **글로벌 레이아웃 (8건)**: 타이포그래피, 디자인 토큰, 헤더, 모바일 네비, 랜딩 페이지, 푸터, 인증 리디자인, 알림 벨
- **제품 마감 (8건)**: 온보딩, SEO, 동적 타이틀, 스켈레톤, 404, 에러 페이지, PWA, 접근성
- **커뮤니티 토론 (6건)**: 모델, CRUD API, 댓글 API, 목록 UI, 작성 폼, 상세+댓글 UI
- **데일리 브리핑 (5건)**: 시장 데이터 수집, AI 생성, API, 대시보드 카드, 아카이브+미국 시장
- **뉴스 피드 (5건)**: 뉴스 수집 파이프라인, AI 요약, API, 피드 UI, 중요도 분류
- **트렌딩 종목 (4건)**: 트렌딩 계산, 인기 종목 API, 위젯, 전체 페이지
- **마켓 캘린더 (5건)**: 데이터 소스, 모델+API, 월간 뷰, 대시보드 위젯, 사용자 종목 하이라이트
- **인프라 & 통합 (4건)**: 중간 E2E, 전체 E2E, 성능+보안, 최종 통합 검증

## Out of Scope
- 실시간 주가 차트 (증권사 앱 역할)
- 모바일 네이티브 앱
- 소셜 로그인
- 토론 좋아요/싫어요, 신고/차단 기능
- 다크 모드 (v2 검토)
- 결제/구독 시스템

## Definition of Done
- All features in features.json are "done"
- All tests pass (including all previous tests)
- No regressions
